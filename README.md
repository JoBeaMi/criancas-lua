# 🌙 Crianças no Mundo da Lua?

**Ferramenta de Rastreio · Atenção e Linguagem**

Joana Miguel & Joana Carvalho
*NID — Núcleo de Investigação e Desenvolvimento*

---

## Sobre

Aplicação de rastreio para educadores e professores, desenhada para apoiar a identificação precoce de dificuldades de **atenção** e de **linguagem** em contexto escolar.

A ferramenta gera automaticamente:
- **Perfil do aluno** (Predominância Atenção / Linguagem / Misto / Baixo Risco)
- **Decisão recomendada** (Encaminhar / Monitorizar / Implementar estratégias)
- **Estratégias práticas** adaptadas ao perfil identificado

## Estrutura

### Parte 1 — O Meu Perfil (~10 min)
- Consentimento informado
- Dados sociodemográficos (anónimos)
- 16 itens Verdadeiro/Falso sobre atenção e linguagem
- Autoeficácia percebida (5 itens)
- Estratégias utilizadas (16 itens)
- 2 vinhetas clínicas (apresentadas oralmente)

### Parte 2 — Rastreio de Alunos
- 36 itens organizados em 6 domínios
- Cálculo automático de perfil e decisão
- Estratégias recomendadas por perfil

### Parte 3 — Painel de Resultados
- Distribuição de perfis
- Médias por domínio
- Resumo dos alunos avaliados

## Utilização

A aplicação funciona inteiramente no browser — sem instalação, sem registo, sem recolha de dados pessoais.

**⚠️ Os dados dos alunos rastreados não são guardados entre sessões. Faça captura de ecrã antes de fechar a aplicação.**

## Tecnologia

Aplicação React standalone (ficheiro único `index.html`), alojada em GitHub Pages. Não requer servidor backend para funcionamento base.
